# Product Overview

WellClinic is a bilingual (English/Arabic) telemedicine platform that connects patients with doctors for online consultations. The system handles doctor profiles, appointment scheduling, payment processing, and Zoom meeting integration for virtual consultations.

## Core Features

-   Doctor directory with specializations and profiles
-   Real-time slot-based appointment booking system
-   Payment processing (Stripe, Paymob, Cash)
-   Zoom meeting integration for virtual consultations
-   Filament-based admin panel for clinic management
-   Multi-language support (English/Arabic) with RTL support

## User Roles

-   **Admin**: Full system access via Filament admin panel
-   **Doctor**: Medical professionals with profiles, availability schedules, and consultation pricing
-   **Patient**: End users who book appointments (no authentication required currently)

## Key Workflows

1. Patients browse doctors by specialization
2. Select available time slots from doctor's calendar
3. Complete booking with contact information
4. Process payment through integrated gateway
5. Receive Zoom meeting link for virtual consultation
